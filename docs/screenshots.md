# Screenshots

> Screenshots will be added after the web part is deployed and tested on a live SharePoint Online site.

## Expected Views

### 1. Main Dashboard View
The full web part showing all sections on a modern SharePoint page.

### 2. Risk Score Card
Close-up of the risk score indicator with score number, progress bar, and key findings.

### 3. External Users Table
The detected external users table with display names, detection reasons, and source groups.

### 4. Permission Inheritance
The permission indicators showing site and library inheritance status.

### 5. Recommended Actions
The contextual governance recommendations panel.

### 6. Property Pane Configuration
The web part settings showing the internal domains configuration field.

### 7. Error / Partial Results
How the web part handles permission errors and shows partial data gracefully.

---

## How to Capture Screenshots

1. Deploy the web part to a test site with the test data provisioned
2. Configure internal domains in the property pane
3. Capture full-page and section-level screenshots
4. Save in this folder as PNG files with descriptive names

Suggested file names:
- `dashboard-full.png`
- `risk-score-card.png`
- `external-users-table.png`
- `permission-indicators.png`
- `recommended-actions.png`
- `property-pane.png`
- `error-state.png`
- `loading-state.png`
